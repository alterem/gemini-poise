# Gemini Poise 前端项目

这是一个基于 React 和 Vite 的前端项目。

## 项目设置

1.  **进入项目目录**:
    ```bash
    cd gemini-poise-frontend
    ```

2.  **安装依赖**:
    使用 pnpm 安装项目依赖。
    ```bash
    pnpm install
    ```

## 运行项目

在开发模式下运行项目：

```bash
pnpm run dev
```

这将在本地启动一个开发服务器，通常在 `http://localhost:5173`。

## 构建项目

构建生产环境的静态文件：

```bash
pnpm run build
```

构建后的文件将位于 `dist/` 目录下。
